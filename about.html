About NeuroLab

NeuroLab is an experimental platform designed to investigate visual working memory, attention, and pattern recognition. The study presents participants with controlled sequences of visual stimuli, progressively increasing in complexity to measure recall accuracy and response time.

Each session adapts dynamically to participant performance, allowing the collection of detailed cognitive metrics across multiple trials. The system records sequence reproduction accuracy, selection timing and error patterns for subsequent analysis.

The experiment is implemented to ensure a consistent and precise environment, facilitating reliable observation of memory processes and attentional control. Data gathered contributes to research in cognitive neuroscience, memory retention and perceptual learning.

*Disclaimer: NeuroLab is a research-oriented platform. All data collected is used for cognitive analysis purposes.*
